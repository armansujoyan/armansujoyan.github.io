ls

List Directory Content

ls [options] [file]

ls options


-a, --all   List all entries including those starting with a dot.
-l   Use a long listing format
-lh   Same as -l, but shows sizes in human readable format.
-F     Add the ‘/’ Character at the end each directory.
-r  Display files and directories in reverse order.
-R  Recursively list Sub-Directories
-ltr Shows latest modification file or directory date as last.
-lS  Displays file size in order, will display big in size first.

ls examples


$ ls ~
$ ls -al
# List the directories in the current directory:
$ ls -d */

grep

Search inside the files

grep [OPTIONS] PATTERN [FILE...]

grep examples


$ grep -B5 "the living" gettysburg-address.txt        # show all matches, and five lines before each match
$ grep -A10 "the living" gettysburg-address.txt       # show all matches, and ten lines after each match
$ grep -B5 -A5 "the living" gettysburg-address.txt    # five lines before and ten lines after
                
$ echo "paranoid schizophrenic walks into a bar" | grep bar

Shortcuts you don't want to forget

  • Ctr+R - Search in previous commands
  • Ctr+L + Clear the Terminal
  • Ctrl+Shift+C - Copy from terminal
  • Ctrl+Shift+V - Paste into terminal
  • Ctrl+W - Cut last word into special clipboard
  • Ctrl+U - Cut the line into special clipboard
  • Ctrl+y - Paste from the same special clipboard
